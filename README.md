# Hello Bricks

Mini site em HTML e CSS desenvolvido como atividade do Transforme-se, com tema inspirado nos jogos de LEGO. O projeto foi pensado para treinar estrutura semântica, navegação entre páginas, formulários e uso de elementos básicos de mídia.
Quis fazer um website que me lembrasse de algo que sempre gostei e me conectasse com a minha infância. Ficou bem simples, mas esse é o objetivo né.

## Sobre o projeto

O site é dividido em três páginas interligadas:

- `index.html`: página inicial com apresentação do tema, imagens, destaques e players de áudio.
- `sobre.html`: página com conteúdo explicativo, títulos hierárquicos, imagem e vídeo incorporado.
- `contato.html`: página com formulário para envio de mensagem.

Todo o visual é controlado por um único arquivo de estilos, `style.css`, que cria uma identidade escura com destaque em amarelo para combinar com o universo LEGO.

## Objetivo acadêmico

Este projeto foi criado para praticar os fundamentos de HTML e CSS, como:

- organização de páginas em múltiplos arquivos;
- uso de links internos;
- estruturação de conteúdo com semântica;
- criação de formulários;
- inclusão de mídia externa.

## Créditos

Projeto desenvolvido para fins de estudo e aprendizagem no programa Transforme-se da Serasa Experian.
